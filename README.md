# 🌟 Growth_Space

**Growth_Space** é uma **landing page elegante e minimalista** inspirada no design futurista do Figma, com **cores brancas e detalhes dourados** ✨. Conta com **animações de astronauta** que dão vida ao layout e criam uma experiência imersiva para o usuário.  

---

## ✨ Destaques do Projeto

- 🪐 **Astronauta Animado:** Vídeo integrado que cria sensação de movimento e imersão.  
- 🎨 **Cores Sofisticadas:** Fundo branco com detalhes em dourado e tipografia elegante.  
- ⚡ **Layout Minimalista:** Foco no conteúdo e na experiência do usuário.  
- 🌐 **Responsivo:** Funciona perfeitamente em qualquer dispositivo.  
- 🎯 **Botões e Call-to-Action:** Estilo clean com hover sutil.  

---

## 💻 Tecnologias Utilizadas

| Tecnologia | Uso |
|-----------|-----|
| HTML5     | Estrutura da página |
| CSS3      | Layout, cores, animações e hover effects |
| JavaScript| Interações e comportamento dinâmico |
| Vídeo Figma | Astronauta animado para destaque visual |
